### Стек:
    - Python 3.10
    - Django 4.1
    - Postgres
    - Django Rest Framework
    - Python Social Auth


### Запуск:
1) Установить зависимости.
2) Заполнить .env следующими значениями:
   * SECRET_KEY
   * DEBUG
   * DB_ENGINE
   * DB_NAME
   * DB_USER
   * DB_PASSWORD
   * DB_HOST
   * DB_PORT
   * DATABASE_URL
   * SOCIAL_AUTH_VK_OAUTH2_SECRET
   * SOCIAL_AUTH_VK_OAUTH2_KEY
3) docker-compose up --build
